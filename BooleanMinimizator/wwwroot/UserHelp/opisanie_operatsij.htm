4 ОПИСАНИЕ ОПЕРАЦИЙ
1. Описание выполняемых функций, задач и процедур
Программный комплекс выполняет следующие функции:
  • Построение таблицы истинности на основании введённого логического выражения или вектора значений;
  • Построение совершенной конъюнктивной нормальной формы (СКНФ) и совершенной дизъюнктивной нормальной формы (СДНФ) по данным таблицы истинности;
  • Формирование карты Карно на основе данных таблицы истинности;
  • Построение минимальной дизъюнктивной нормальной формы (МДНФ) и минимальной конъюнктивной нормальной формы (МКНФ);
  • Визуализация процесса минимизации с пошаговым выделением смежных областей на карте Карно;
  • Ввод логического выражения с помощью встроенной клавиатуры, поддержка логических операций: ∨, ∧, ¬, ⊕, →, ↔️, а также переменных w, x, y, z.
    2. Описание операций обработки данных
Операция 1: Ввод логического выражения
  •  Наименование: Ввод логического выражения с помощью виртуальной клавиатуры.
  •  Условия выполнения: Пользователь открыл страницу калькулятора, доступна форма ввода.
  •  Подготовительные действия: Ознакомление с синтаксисом логических операций. Выбор переменных (w, x, y, z).
  •  Основные действия:
  • Нажимать кнопки виртуальной клавиатуры для составления выражения.
  • Использовать скобки для задания приоритетов операций.
  • При необходимости ввести вектор функции (например: 10011010).
    5. Заключительные действия: Нажать кнопку «Построить».
Операция 2: Построение таблицы истинности
1. Наименование: Генерация таблицы истинности по введённому выражению или вектору.
2. Условия выполнения: Введено корректное логическое выражение или вектор длины 2^n.
3. Подготовительные действия: Парсинг выражения, построение дерева разбора.
4. Основные действия:
  • Генерация всех комбинаций переменных.
  • Вычисление значений функции для каждой строки.
  • Отображение таблицы на экране.
    5. Заключительные действия: Вывод готовой таблицы с переменными и значениями.
Операция 3: Построение карты Карно
1. Наименование: Формирование и отображение карты Карно.
2. Условия выполнения: Построена таблица истинности.
3. Подготовительные действия: Определение количества переменных благодаря введённой пользователем функции.
4. Основные действия:
  • Распределение значений из таблицы по карте.
  • Отображение карты с подписями переменных.
    5. Заключительные действия: Карта визуализирована на экране.
Операция 4: Построение начальных МДНФ и МКНФ
1. Наименование: Автоматическое построение полного МДНФ и МКНФ по таблице.
2. Условия выполнения: Таблица истинности готова.
3. Подготовительные действия: Выделение строк, где функция равна 1 (для МДНФ) или 0 (для МКНФ).
4. Основные действия:
  • Генерация конъюнкций (для МДНФ) или дизъюнкций (для МКНФ).
  • Составление общей формулы.
    5. Заключительные действия: Вывод выражений на экран.
Операция 5: Минимизация и визуализация на карте Карно
1. Наименование: Пошаговая визуализация минимизации с выделением областей на карте Карно.
2. Условия выполнения: Карта Карно построена.
3. Подготовительные действия: Анализ карты, определение возможных групп (1, 2, 4, 8…).
4. Основные действия:
  • Последовательно выделяются группы единиц.
  • Формируются минимальные термы.
  • Обновляется выражение МДНФ.
    5. Заключительные действия: Вывод минимизированного МДНФ (затем МКНФ) на экран.
